GIF-a-tron

Brief

Build an app utilizing AJAX and Giphy's API to load animated GIFs based on specific keywords.

Additional info

App was built and tested locally, version-controlled with GitHub, and finally deployed on GitHub.

Check out the live app!

Click any of the preloaded action buttons

Animated GIFs based on that keyword will load

Type in your own action keyword

A new set of animated GIFs will load and a new button will be added at the top of the page
